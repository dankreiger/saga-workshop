## Redux and Redux-Saga workshop

### Getting started

```sh
$ nvm install 12.16.1
$ nvm use
$ yarn
$ yarn start
```

For ex5 please do the following:

```sh
$ cd src/ex5
$ nvm use
$ yarn
$ yarn start
$ yarn test # to run specs
```

For ex6 please do the following:

```sh
$ cd src/ex6
```
